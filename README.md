# IPTV Script
Makes USA IPTV lists automagically, list contents brought to you by our comrades at @Free-TV and @iptv-org.

## How to
```bash
curl "https://raw.githubusercontent.com/MyMel2001/iptv-script/refs/heads/main/generate.sh" | bash
```

## Want an easy way to have it automated?

Use [this GitHub action here.](https://github.com/mymel2001-holder/iptv-action)
